# 셀러 마진 리포트 자동화 대시보드

## 한 줄 요약

스마트스토어·쿠팡 CSV와 상품 원가표를 합쳐 상품별 순이익, 저마진 상품, 원가 누락 경고를 자동으로 보여주는 온라인 셀러용 정산·마진 분석 데모입니다.

## 프로젝트 목적

온라인 셀러는 플랫폼별 주문/정산 파일, 원가표, 배송비, 수수료 기준을 따로 관리하는 경우가 많습니다. 이 프로젝트는 반복 엑셀 작업을 줄이고, 매출이 아니라 실제 남는 돈 기준으로 상품을 판단할 수 있게 만드는 웹 대시보드형 포트폴리오입니다.

## 대상 고객

- 스마트스토어, 쿠팡, 자사몰을 함께 운영하는 1인 사업자
- 엑셀로 정산 파일을 매번 합치고 원가를 대조하는 소상공인
- 매출은 확인하지만 상품별 실제 순이익은 늦게 파악하는 온라인 셀러

## 해결하는 문제

- 플랫폼마다 다른 CSV 컬럼을 수작업으로 정리해야 합니다.
- 원가, 수수료, 배송비를 반영한 순이익 계산이 번거롭습니다.
- 저마진/적자 상품과 원가 누락 주문을 늦게 발견합니다.
- 월별, 플랫폼별 흐름을 엑셀 표만으로 파악하기 어렵습니다.

## MVP에서 확인할 수 있는 가치

- 샘플 CSV 4종만으로 전체 자동화 흐름을 바로 시연할 수 있습니다.
- 상품별 순이익, 마진율, 적자/저마진 상태가 한 화면에서 정리됩니다.
- 원가 누락, 배송비 누락 의심, 수수료 규칙 누락을 경고 패널과 엑셀 리포트에 함께 반영합니다.
- 크몽 의뢰 시 고객 CSV 양식에 맞춰 어떤 자동화가 가능한지 설명하기 쉽습니다.

## 주요 기능

- 스마트스토어·쿠팡 CSV 표준화
- 상품 원가표와 플랫폼 수수료표 기준 마진 계산
- 총매출, 순이익, 평균 마진율, 경고 건수 KPI
- 월별 매출·순이익 차트
- 플랫폼별 매출·순이익 비교 차트
- 상품별 마진 분석 테이블
- 원가 미등록, 배송비 누락 의심, 적자/저마진 상품 경고
- 엑셀 리포트 다운로드

## 실행 방법

```bash
cd app
npm ci
npm run dev
```

브라우저에서 `http://localhost:3000?sample=1`로 접속하면 샘플 데이터가 자동 적용됩니다.

## 테스트 방법

```bash
cd app
npm test
npm run lint
npm run build
npm audit --audit-level=high
```

현재 MVP 검수 기준은 단위 테스트, 린트, 프로덕션 빌드, high 등급 보안 취약점 없음입니다. `npm audit`의 moderate 항목은 현재 자동 수정이 breaking downgrade라 유지합니다.

## 데모 흐름

1. `샘플 데이터로 체험하기`를 클릭하거나 `?sample=1` 주소로 접속합니다.
2. 주문 CSV, 원가표, 수수료표가 대시보드에 반영됩니다.
3. KPI와 차트로 전체 흐름을 확인합니다.
4. 상품별 마진 테이블에서 적자/저마진 상품을 확인합니다.
5. 경고 패널에서 원가 누락과 검증 대상을 확인합니다.
6. 엑셀 다운로드 버튼으로 리포트를 저장합니다.

## 실제 의뢰 적용 방식

1. 고객의 실제 CSV 양식과 정산 기준을 확인합니다.
2. 플랫폼별 컬럼명을 표준 주문 데이터로 매핑합니다.
3. 원가표, 배송비, 수수료 계산 규칙을 고객 업무 기준에 맞춥니다.
4. 대시보드 항목과 엑셀 리포트 양식을 조정합니다.
5. 고객이 반복 사용할 수 있도록 실행 방법과 파일 입력 규칙을 문서화합니다.
