# 셀러 마진 리포트 자동화 대시보드

## 프로젝트 목적

스마트스토어와 쿠팡 주문 CSV, 상품 원가표, 수수료 설정표를 한 번에 합쳐 온라인 셀러가 상품별 순이익과 적자 상품을 빠르게 확인할 수 있는 웹 대시보드형 포트폴리오 데모입니다.

## 대상 고객

- 스마트스토어, 쿠팡, 자사몰을 함께 운영하는 1인 사업자
- 엑셀로 정산 파일을 매번 합치고 원가를 대조하는 소상공인
- 매출은 보지만 상품별 실제 순이익을 늦게 확인하는 온라인 셀러

## 해결하는 문제

- 플랫폼마다 다른 CSV 컬럼을 수작업으로 정리해야 합니다.
- 원가, 수수료, 배송비를 반영한 순이익 계산이 번거롭습니다.
- 저마진/적자 상품과 원가 누락 주문을 늦게 발견합니다.
- 월별, 플랫폼별 흐름을 엑셀 표만으로 파악하기 어렵습니다.

## 주요 기능

- 스마트스토어·쿠팡 CSV 표준화
- 상품 원가표와 플랫폼 수수료표 기준 마진 계산
- 총매출, 순이익, 평균 마진율, 경고 건수 KPI
- 월별 매출·순이익 차트
- 플랫폼별 매출·순이익 비교 차트
- 상품별 마진 분석 테이블
- 원가 미등록, 배송비 누락 의심, 적자/저마진 상품 경고
- 엑셀 리포트 다운로드

## 사용 기술

- Next.js 16
- TypeScript
- React
- Recharts
- Papa Parse
- ExcelJS
- Vitest

## 데모 흐름

1. `샘플 데이터로 체험하기`를 클릭합니다.
2. 주문 CSV, 원가표, 수수료표가 대시보드에 반영됩니다.
3. KPI와 차트로 전체 흐름을 확인합니다.
4. 상품별 마진 테이블에서 적자/저마진 상품을 확인합니다.
5. 경고 패널에서 원가 누락과 검증 대상을 확인합니다.
6. 엑셀 다운로드 버튼으로 리포트를 저장합니다.

## 제외 범위

- 실제 스마트스토어·쿠팡 API 연동
- 로그인 자동화
- 실시간 주문 수집
- 세무 신고 수준 회계 처리
- 알림톡/SMS 발송

## 확장 가능 항목

- 고객 실제 CSV 양식 맞춤 매핑
- 자사몰/오픈마켓 추가
- 원가표 관리 화면
- 월별 자동 리포트 생성
- 구글시트/노션/슬랙 연동
