# 크몽 상세페이지 초안

## 대표 문구

스마트스토어·쿠팡 주문 CSV를 상품별 마진 리포트로 자동 정리해드립니다.

## 문제 제기

온라인 판매자는 플랫폼마다 다른 주문/정산 파일을 내려받아 엑셀에서 다시 정리하는 일이 많습니다. 매출은 확인하기 쉽지만, 상품별 원가와 수수료, 배송비까지 반영한 실제 순이익은 늦게 파악되는 경우가 많습니다.

## 해결 방식

고객의 주문 CSV와 상품 원가표, 플랫폼 수수료 기준을 표준 데이터로 통합한 뒤 상품별·월별·플랫폼별 마진을 계산합니다. 결과는 웹 대시보드와 엑셀 리포트로 확인할 수 있게 구성합니다.

## 제공 결과물

- 주문 CSV 표준화 로직
- 상품 원가표 매칭
- 플랫폼 수수료 계산
- 상품별 순이익/마진율 분석
- 적자·저마진·원가 누락 경고
- 월별/플랫폼별 요약 대시보드
- 엑셀 리포트 다운로드

## 고객 준비물

- 최근 주문 또는 정산 CSV
- 상품별 원가표
- 플랫폼별 수수료 기준
- 원하는 리포트 항목

## 패키지 구성

| 패키지 | 구성 | 기간 |
|---|---|---|
| BASIC | 엑셀/CSV 1종 정리, 기본 계산 자동화 | 2~3일 |
| STANDARD | 쇼핑몰 CSV 1~2종 통합, 마진 계산, 요약 대시보드 | 5~7일 |
| PREMIUM | 다중 플랫폼 통합, 엑셀 리포트, 오류 검증, 사용 가이드 | 10~14일 |

## 추가 맞춤 개발 가능 항목

- 고객사 CSV 양식 맞춤 매핑
- 자사몰·기타 오픈마켓 추가
- 구글시트 자동 업데이트
- 월별 리포트 자동 생성
- 슬랙/이메일 알림 연동

## 포트폴리오 데모 기준

이 포트폴리오는 실제 플랫폼 API 연동 없이 샘플 CSV로 자동화 흐름을 보여주는 데모입니다. 실제 의뢰에서는 고객 파일 양식과 업무 규칙을 확인한 뒤 맞춤형으로 조정합니다.
